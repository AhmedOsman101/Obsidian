# Java Programming Task 5

## Question 1

Write a Java program to find the common elements between two arrays of integers.

## Question 2

Write a Java program to find the second largest element in an array.

## Question 3

Write a method that shifts all the elements in the array one position to the right and moves the last element to become the first element in the array.

## Question 4

Write a Java program to compute the average value of an array of integers except the largest and smallest values.

## Question 5

Write a Java program to arrange the elements of a given array of integers where all positive integers appear before all the negative integers.

## Question 6

Write a Java program to count repeated elements in an array.

## Question 7

Remove odd elements from the array `{11,12,13,14,15}` and print it.

### Output

`{12,14}`

## Question 8

Merge two arrays `arr1{30,10,50}` and `arr2{5,100,70}`. The merged array should be `{30,10,50,5,100,70}`, then pass the merged array to a function to sort it in ascending order.

## Question 9

Create an array of size `10`, fill it with random numbers from `10` to `45`, then send the array to a function called `sort` to sort the array in descending order and print the array.

## Question 10

Given an array `{10,20,30,40}`, insert an element `100` in position `2`.

### Output

`{10,20,100,30,40}`
